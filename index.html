How The TikTok Flow Works

Step 1

Login entry point

Creators use the visible Creator Login button to start TikTok OAuth from this official website.

Step 2

Creator approval

TikTok prompts the creator to authorize Eyouel Exchange for the Login Kit and Content Posting API upload scopes.

Step 3

Content posting

The creator reviews the video and metadata, confirms TikTok notices, and then uploads the draft through the secure backend.

Reviewer Information

Redirect URI for the web configuration: https://eyouel-cloud.github.io/auth/tiktok/callback.html. Requested TikTok scopes: user.info.basic and video.upload. Client secret and refresh tokens are handled only by the server-side upload service and are not embedded in this public website.
